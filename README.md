# AirPump

The purpose of this project is to control a DC air pump with a L293D motor driver, and display 
the temperature of a fish tank in a LCD display.

## Getting Started

Materials:
- Arduino
- 10k Thermistor
- 10k Resistor
- LCD Display
- Potentiometer
- DC Air pump
- L293D Motor Driver
- 100uF capaciton
